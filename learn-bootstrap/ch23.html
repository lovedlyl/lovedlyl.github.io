23.1 实现过程

  1. 轮播图片采用.carousel样式，并且需要一个ID方便以后使用data属性触发。
    <div id="slidershow" class="carousel"></div>
  2. 轮播图片计数器采用类.carousel-indicators一般为有序列表ol，有几张图片就有几个li元素，需要默认显示的图片对应的li元素需要添加类.active
     <div id="slidershow" class="carousel">
	<ol class="carousel-indicatiors">
		<li class="active"></li>
		<li></li>
	</ol>
</div>
  3. 图片区使用类.carousel-inner并且使用.item放置每个图片，如果需要默认显示，还需追加类.active 
     <div id="slidershow" class="carousel">
	<ol class="carousel-indicatiors">
		<li></li>
		<li></li>
	</ol>
	<div class="carousel-inner">
		<div class="item active"><a href="#"><img src="*.jpg" alt=""></a></div>
		<div class="item"><a href="#"><img src="*.jpg" alt=""></a></div>
	</div>
</div>
  4. 如果需要在图片上添加标题和描述内容，还可以在.item内添加.carousel-caption
     <div class="item active">
	<a href="#"><img src="1.jpg" alt=""></a>
	<div class="carousel-caption">
		<h4>标题</h4>
		<p>描述内容</p>
	</div>
</div>
  5. 实现控制器使用<a>元素，使用类.carousel-control并结合类.left 和.right。箭头图标使用.glyphicon.glyphicon-chevron-left/right或.icon-prev/nect
    <div id="slidershow" class="carousel">
	<ol class="carousel-indicatiors">
		<li></li>
		<li></li>
	</ol>
	<div class="carousel-inner">
		<div class="item"><a href="#"><img src="*.jpg" alt=""></a></div>
		<div class="item"><a href="#"><img src="*.jpg" alt=""></a></div>
	</div>
	<a href="#" class="carousel-control left"><span class="icon-prev"></span></a>
	<a href="#" class="carousel-control left"><span class="icon-next"></span></a>
</div>

  6. 控制轮播的高度:需要同时设置.carousel和.carousel-item的高度为一样。并且还要设置图片宽度为100%。

    .carousel, .carousel .item{
	height: 500px;
	bacground-color: #000;
}
.carousel img{
	width: 100%;
}

    响应式更改：当屏幕缩小时，为防止图片不能填满整个容器，还需要响应式修改：

    @media(max-width: 768px){
	.carousel{
			margin-bottom: 30px;
	}
	.carousel, .carousel .item{
		height: 300px;
		bacground-color: #000;
	}
	.carousel img{
		min-height: 300px;
	}	
	.carousel .carousel-caption p{
		font-size: 16px;
		line-height: 1.4;
	}

23.2 data-属性触发

上一节只是实现了轮播图片的基本结构，如果要通过属性触发，还要：

  1. 为.carousel添加属性data-ride="carousel"。如果需要图片切换更平滑，还可追加类.slide
  2. 计数控制器上的li元素添加属性data-target=".carousel的ID值"，以及data-slide-to属性，取值自身在ol的索引，基数为0。
  3. a元素控制器的href设置为".carousel的ID值"，data-slide属性设置为"next"或"prev"，分别代表向前和向后。
 <div id="slidershow" class="carousel slide" data-ride="carsouel">
	<ol class="carousel-indicatiors">
		<li data-target="#slideshow" data-slide-to="0"></li>
		<li data-target="#slideshow" data-slide-to="0"></li>
	</ol>
	<div class="carousel-inner">
		<div class="item"><a href="#"><img src="*.jpg" alt=""></a></div>
		<div class="item"><a href="#"><img src="*.jpg" alt=""></a></div>
	</div>
	<a href="#slideshow" class="carousel-control left" data-slide="prev"><span class="icon-prev"></span></a>
	<a href="#slideshow" class="carousel-control right" data-slide="next"><span class="icon-next"></span></a>
</div>

参数：

以下参数可以在.carousel上声明，效果相同，data-"参数"=？

名称类型默认值描述
intervalnumber5000图片切换间隔的时间，如果为false则不自动循环
pause stringhover 鼠标在图片上有何种事件是停止播放
wrap布尔值true轮播是否保持循环，即播放一轮后是否重复
