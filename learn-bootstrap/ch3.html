第三章 表单

3.1 表单基础

表单中常见的元素包括：

  • 文本输入框
  • 下拉选择框
  • 单选按钮
  • 复选按钮
  • 文本域
  • 按钮

对于<fieldset>，<legend>，<lable>元素，bootstrap对margin，padding和border做了细化设置。
对于<input>，<select>，<textarea>等元素， 如果加上类名.form-control就会产生定制效果：

  1. 宽度变为100%
  2. 设置一个浅灰色（#ccc）边框
  3. 具有4px的圆角
  4. 设置阴影效果，并且元素得到焦点时，阴影和边框效果都有所变化
  5. placeholder 的颜色为#999

3.2 水平表单

bootstrap表单默认为垂直风格，要实现水平需满足以下条件:

  1. 在<form>元素使用类.form-horizontal
  2. 配合bootstrap网格系统（后面有介绍）

添加类.form-horizontal有以下效果

  1. 设置表单的padding和margin值
  2. 改变.form-group的表现形式，类似于网格系统的"row"
  3. 只有当屏幕宽度大于786px时才能达到此效果

3.3 内联表单

效果：表单所有控件都在一行内显示。 类：.form-inline

注意

  1. <lable>要放置在.form-group内
  2. <lable>元素加上类.sr-only不会显示，放置的目的是方便屏幕阅读器。

3.3 表单控件（输入框 input）

需要指定input元素的type类型，如果没有，不能得到正确样式。

为了让控件在各种表单风格样式不出错，需要添加类名.form-control。

3.5 表单控件（下拉选择框select）

<select>添加类名.form-control包裹在类为.form-group的<div>中。 multiple选项保持不变，自定义。

3.6 表单控件（文本域textarea）

设置rows可定义高度，设置cols可定义宽度。 不过如果在<textarea>添加类.form-control则无需设置cols，bootstrap将其宽度设置为100%或auto

3.7 表单控件（复选框chekbox和radio）

checkbox和radio与其label的对齐很头疼， 这在bootstrap中得到了解决， 实现步骤：

  1. 不管checkbox还是radio都用label包裹起来
  2. checkbox连同label标签放在名为.checkbox的容器内
  3. radio连同label标签放在名为.radio的容器内。 （bootstrap主要借助.checkbox和.radio来处理对齐方式。）

3.8 表单控件（复选框和单选按钮水平排列）

bootstrap实现复选框和单选框水平排列原理：

  1. 如果checkbox需要水平排列，只需在label标签上添加类.checkbox-inline
  2. 如果要radio水平排列，只需在label标签上添加类.radio-inline

3.9 表单控件（按钮）

见第四章 按钮

3.10 表单控件大小

高度控制：可以通过设置height，line-height，padding和font-size 控制控件高度。 不过bootsrtrap提供了两个类名来控制控件高度：

  1. .input-sm：比正常大小更小
  2. .input-lg：比正常大小更大

这两个类名仅适用于<input>，<select>和<textarea>控件。



宽度控制：借助bootstrap的网格系统，可设置控件的宽度。

  1. 方法1：from.form-horizotal[role="form"]>div.form-group>div.col-?-?>input.form-control 即使用表单的.form-horizontal形式，通过在.form-group中的包裹div设置网格系统实现控制控件宽度。
  2. 方法2：div.row>div.col-?-?>input.form-control 即.form-group相当于网格系统中的.row。


3.11 表单控件状态（聚焦状态）

使用bootstrap时，要让表单控件出现焦点效果，需要在控件上添加类.form-control

实现原理：为焦点状态添加伪类:focus

file，radio，checkbox控件与普通input控件不一样，不需要加.form-control类。源码：

input[type="file"]:focus,
input[type="radio"]:focus,
input[type="checkbox"]:focus {
outline: thin dotted;
outline: 5px auto -webkit-focus-ring-color;
outline-offset: -2px;
}

3.13 表单控件状态（验证状态）

bootstrap为表单验证状态提供了下面效果：

  1. .has-warning：警告信息（黄色）
  2. .has-error：错误信息（红色）
  3. .has-success：成功状态（绿色）

使用时，是在.form-group上添加相应状态类名。并且相应的lable元素要添加类.control-label

如果要想在提示时在input框内加上图标，还需要在.form-group后面加上类.has-feedback
并且在input后需加上一个span元素，其类名为.glyphicon.glyphicon-?.form-control-feedback其中?号代表：

  • .glyphicon-ok：成功
  • .glyphicon-warning-sign ：警告
  • .glyphicon-remove：错误
