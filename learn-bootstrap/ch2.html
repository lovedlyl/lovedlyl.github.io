    2.17 表格

    bootstrap为表格提供了1种基础样式，4种附加样式，和1种响应式样式，对应的类为：

    1. .table: 基础表格
    2. .table-striped: 斑马线表格
    3. .table-bordered: 带边框表格
    4. .table-hover: 鼠标悬停高亮表格
    5. .table-condensed: 紧凑型表格
    6. .table-responsive: 响应式表格

    2.18 表格-表格行的类

    bootstrap还为表格行元素<tr>提供了不同类，控制不同行的颜色：

    1. .active
    2. .success
    3. .info
    4. .warning
    5. .danger
    类名描述对应颜色
    .active表示当前活动的信息#f5f5f5
    .success表示成功或者正确的行为#dff0d8
    .info表示中立的信息或者行为#d9edf7
    .warning表示警告，需要特别注意#fcf8e3
    .danger表示危险或者可能错误的行为#f2dede

    2.30 显示和隐藏

    .hidden，.show，.sr-only和.invisible都可控制元素显示和隐藏，其中：

    • .sr-only是指只对屏幕阅读器起作用。
    • .invisible的效果是，虽然元素不可见，但仍占据文档流，影响布局。
       .invisible{
	visibility: hidden
}

    图片替换：可以用类.text-hide或mixin实现

    <div class="text-hide">logo 标题</div>
    .heading {
	.text-hide();
}
