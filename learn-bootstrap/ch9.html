第九章 导航条

9.1 与导航的区别

上一章介绍的导航与本章介绍的导航条有明显的区别，导航条的特点有：

  1. 有统一的背景色
  2. 可以是纯连接（类似于导航），也可以是表单
  3. 也可以将导航和表单结合使用

9.2 基础导航条

制作基础导航条的步骤是：

  1. 在导航列表(<ul class="nav">)的基础上增添类.navbar-nav
  2. 列表外部添加容器(div)并使用类.navbar和.navbar-default

  • 默认
  • 激活
  • 禁用

    9.4 带表单的导航条

    导航条中还可包含表单，实现方法：

    1. 为<form>元素添加类.navbar-form
    2. 如果需要表单靠近其左端内容，可添加类.navbar-left。（默认状态）
    3. 如果需要靠近最右端内容，可添加类.navbar-right
    4. 上面“左右”靠近方式同样适用于其它导航条元素

    9.6 将导航条固定在浏览器顶部和底部

    实现方法：在.navbar-default基础上添加类.navbar-fixed-bottom或.navbar-fixed-top：演示页面

    实现原理：将导航条设置为position:fixed，top和bottom值设置为0。

    可以观察到页面底部和顶部内容被导航条遮住了，导航条默认高度为50px，所以解决方法有两种

    1. 在body上设置padding
       body {
	padding-top: 70px;/*有顶部固定导航条时设置*/
	padding-bottom: 70px;/*有底部固定导航条时设置*/
}
      2. 演示页面
    2. 将固定导航条放置在页面内容前（~选择器：匹配元素之后的同级元素）
       .navbar-fixed-top ~ .content {
	padding-top: 70px;
}
.navbar-fixed-bottom ~ .content {
	padding-bottom: 70px;
}
      3. 演示页面

    9.7 响应式导航条

    当屏幕宽度<768px时，整个导航内容隐藏，出现按钮，点击按钮时导航内容出现

    实现方法：

    1. 需要折叠的内容放在.collapse.navbar-collapse.navbar-responsive中。并为其添加一个.类名或#id
    2. 将以下内容放置在.navbar-header中（固定写法）
       <button class="navbar-toggle" type="button" data-toggle="collapse">
	<span class="sr-only">Toggle Navigation</span>
	<span class="icon-bar"></span>
	<span class="icon-bar"></span>
	<span class="icon-bar"></span>
</button>
    3. 为上面的button添加属性data-target=".class/#id"。这里的id或class由需要折叠的内容上添加的id或class决定。
