第二十一章 按钮（JS）

这里讨论的不是按钮组件的制作，而是通过按钮实现：

  1. 控制按钮状态，比如禁用状态，正在加载，正常状态。比如，当点击按钮时出现”正在加载...“，加载完成后显示”获取成功“
  2. 切换按钮状态
  3. 模仿单选/复选按钮
在页面多次加载之间，Firefox 仍然保持表单控件的状态（禁用状态和选择状态）。一个解决办法是设置 autocomplete="off"

21.1 按钮加载状态

效果：点击按钮时显示”正在加载...“的提示信息

实现：

  1. 为button元素添加data-loading-text属性，值为处于加载状态时按钮的文本内容
  2. 调用button的button("loading")方法
 <button class="btn btn-primary" id="btn1" type="button" auto-complete="off" data-loading-text="正在加载，请稍等...">加载</button>
<script>
$("#btn1").click(function() {
	$(this).button("loading")
})

从 v3.3.5 版本开始，此特性不再建议使用，并且将在 v4 版本中删除。

基本结构：.btn-group[data-toggle="buttons"] > label.btn.btn-primary > input[type="radio"....] 文本

21.4 JS调用

按钮可以使用button()方法的3种调用方式：

  1. 切换按钮状态（得到焦点） ：
    $("#myButton").button("toggle")
  2. 重新恢复按钮：
    $("#myButton").button("reset")
     <button class="btn btn-primary" type="button" data-toggle="button" data-loading-text="正在加载..." auto-complete="off" id="btn2">点我加载</button>
<script>
$("#btn2")
.click(function() {
	$(this).button("loading")
		.delay(1000)
		.queue(function() {
			$(this).button("reset")
		})
})
</script>
  3. 任意参数
    $("#myButton").button("任意参数")
    在元素上设置属性data-任意参数-text，就可在调用button方法后使按钮文本显示为属性设置值。
     <button class="btn btn-primary" type="button" auto-complete="off" data-toggle="button" data-loading-text="正在加载..." data-complete-text="加载完成" id="btn4">点我加载 </button>
<script>
   $("#btn4").click(function() {
       $(this).button("loading").delay(1000).queue(function() {
           $(this).button("complete")
       })
   })
</script>
